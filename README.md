# HydraVisual

🐉 Визуальный клиент для Minecraft 1.21.4

## Модули

| Модуль | Описание | Клавиша |
|--------|----------|---------|
| Fullbright | Полная яркость | Right Shift |
| FPS | Отображение FPS | Alt + H |
| Coords | Координаты XYZ | Alt + H |

## Установка

1. Установите Fabric для Minecraft 1.21.4
2. Скачайте последний релиз из [Releases](https://github.com/sigmasvinkak-cmyk/hydravisual/releases)
3. Поместите `.jar` в папку `mods/`
4. Или используйте **HydraVisual Launcher** для автоматической установки!
